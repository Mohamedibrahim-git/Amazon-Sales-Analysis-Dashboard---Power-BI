Amazon Power BI Dashboard 📊

📌 Project Overview

This project contains a Power BI dashboard built to analyze Amazon sales data. It provides insights into sales performance, product categories, customer behavior, and key business metrics.

 📁 File Included

`amazon power bi Dashboard.pbix` – Main Power BI dashboard file

🚀 Features

* Interactive dashboard with filters and slicers
* Sales analysis by category, region, and time
* Key performance indicators (KPIs) visualization
* Trend analysis using charts and graphs

🛠️ Tools & Technologies

* Power BI
* Data Visualization
* Data Analysis

📊 Insights

* Identify top-performing products
* Analyze revenue trends
* Understand customer purchase patterns

▶️ How to Use

1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore the dashboard using filters and visuals

📌 Future Improvements

* Add real-time data integration
* Include advanced analytics (forecasting)
* Enhance UI/UX of dashboard
